# Citing

If you would like to cite the `gorder` software, use the following BibTeX entry:

```text
@software{gorder,
  author       = {Ladislav Bartos},
  title        = {gorder},
  month        = dec,
  year         = 2024,
  publisher    = {Zenodo},
  doi          = {10.5281/zenodo.14391305},
  url          = {https://doi.org/10.5281/zenodo.14391305}}
```
